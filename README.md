# Weather Condition Checker

## Description
This repository contains a simple PHP script that determines the weather condition based on temperature.

## Features
- Determines weather conditions (cok soguk, soguk, serin, sicak, cok sıcak) based on the given temperature.
- User-friendly HTML interface to display the weather condition.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/S-cakl-k-php-if-yap-s-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd S-cakl-k-php-if-yap-s-
   ```

## Usage
1. Open the `index.php` file in your web browser.
2. The script will display the weather condition based on the predefined temperature.

## Tech Stack
- PHP
- HTML
